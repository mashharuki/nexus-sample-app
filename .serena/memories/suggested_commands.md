# 推奨コマンド
- `bun install` — 依存関係をインストールします (利用するパッケージマネージャーは Bun)。
- `bun run start` / `bun run dev` — Vite 開発サーバーをポート 3001 で起動します。
- `bun run build` — Vite で本番ビルドを行い、その後 `tsc` で型チェックします。
- `bun run serve` — ビルド済みアセットをローカルでプレビューします。
- `bun run test` — Vitest のテストスイートを実行します。
- `bun run lint` — ESLint によるチェックを実行します。
- `bun run format` — Prettier でコードをチェック/整形します (デフォルトでは書き込みなし)。
- `bun run check` — Prettier と ESLint の修正を自動適用します (ファイルへ書き込みが発生)。
- macOS 基本操作: `ls`, `pwd`, `rg`, `bun`, `git` などを zsh シェルから利用可能です。
